how to use

if you are here for kiss ultra v2 settings, scroll down to the ultra-v2-tuning section.

NOTE : v1 users ought to use the performance settings and tpa pictured in the v2 section

NOTE2: pid presets can be opened by holding select on pid profile page

Use the following PID's as 3 step process to triangulate the best RANGE for your quad.

  1. start at the ULTRA defaults -- should fly great on almost any quad
  2. try the high D -- if it TRILLS or makes bad noises, you know that high D is too high
  3. try the low D -- if the quad bounces back harshley or has bounceback, you now know your quad will want more D.

ultra-v2-tuning

  1. kiss ultra v2, on the latest fw
  2. experimental mode must be ON
  3. TPA setting is changed to give authority back at low throttle and kill some mid throttle oscillations
  4. Flight performance tab maxed out -- becuase we do all the things ultra
  5. if you have issues with these settings, return to ultra defaults

ADVANCED TUNING NOTE:
you can use higher pd gain's than pictured, remember this is a 'golden' tune.. use the below numbers as a starting point and use the ratio of 1p:3.5d, and you'll find that on the cleanest builds youll be able to push the pd gains up near a 2.0 multiplyer; your mileage may vary based on a lot of factors so raise your pd gains from the golden starting point at your own risk
